# ZZ 打票 (ZZ-Printer) 🚀

**ZZ 打票**是一款专为财务、个人报销场景设计的发票拼版工具。它能够将分散的 PDF/OFD 电子发票高效地合并在 A4 纸上，支持多种排版布局，完美适配 Windows 深色模式。

---

## 📖 功能亮点

* **双格式支持**：全面兼容传统 **PDF** 及新版国标 **OFD** 发票格式。
* **智能拼版布局**：
* `1×2`：单列纵向排列（两张发票上下分布）。
* `2×3` / `2×4`：双列纵向排列（每页可放 6-8 张发票）。


* **iOS 风格 UI**：简洁灵动的界面，支持随系统自动切换的**深色/浅色模式**。
* **交互效率**：支持**文件拖拽**、**路径记忆**、**批量多选移除**。

## 🛠️ 环境部署与安装

如果你是开发者或希望从源码运行，请参考以下步骤：

### 1. 前置要求

确保你的电脑已安装 **Python 3.8** 或更高版本。

### 2. 获取代码

```bash
git clone https://github.com/your-username/zz-printer.git
cd zz-printer

```

### 3. 创建虚拟环境（强烈推荐）

为避免库冲突，建议使用独立环境：

```powershell
# 创建虚拟环境
python -m venv venv

# 激活环境 (Windows)
.\venv\Scripts\activate

```

### 4. 安装依赖

```powershell
pip install -r requirements.txt

```

## 🚀 使用指南

### 第一步：添加文件

* **点击按钮**：使用“添加文件”或“文件夹导入”选择发票。
* **拖拽操作**：直接从资源管理器将 PDF/OFD 文件甩进程序窗口。

### 第二步：参数配置

* **打印模式**：选择“打印为 PDF”（推荐，可保存后查看）或“直接打印”。
* **页面布局**：根据需要选择排版（例如：`1×2` 适合大图预览，`2×4` 适合节省纸张）。
* **打印份数**：设置每张发票重复出现的次数。

### 第三步：列表管理

* **多选移除**：在列表中配合 `Ctrl` 或 `Shift` 键点击文件名，点击“移除选中”即可。
* **实时预览**：右侧区域会根据你的配置实时更新拼版效果。

## 📦 打包发布

如果你想将程序打包给没有 Python 环境的同事使用，请运行我们提供的自动化脚本：

1. 在项目根目录找到 `build.bat`。
2. **双击运行**。
3. 脚本会自动创建纯净环境并完成打包。
4. 完成后，在 `dist\ZZ打票` 文件夹下即可找到 `ZZ打票.exe`。

> **注意**：分发时请务必将整个 `ZZ打票` 文件夹压缩发送，不要只发单个 `.exe` 文件，以确保启动速度。
